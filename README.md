# Sales Power BI Dashboard

## 📊 Overview
This repository contains **First Report.pbix**, a structured Power BI dashboard developed to provide interactive data insights. The report features a custom-designed layout and a robust underlying data model to facilitate clear data storytelling.

## 🛠️ Technical Components
The dashboard is built with the following core components:
* **Data Engine**: A comprehensive `DataModel` that handles table relationships and calculations.
* **Visual Theme**: Uses the `CY23SU04.json` base theme for a professional and modern aesthetic.
* **Layout Design**: A custom `DiagramLayout` and `Report/Layout` configuration optimized for user experience.
* **Static Resources**: Includes custom registered artwork and branding elements.
* **Security**: Integrated `SecurityBindings` to manage data access and integrity.

## 🚀 Getting Started
1. **Download the File**: Clone this repository or download the `First Report.pbix` file directly.
2. **Software Requirements**: Install the latest version of [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. **Open the Report**: Launch the file to explore the interactive visualizations and data schema.
